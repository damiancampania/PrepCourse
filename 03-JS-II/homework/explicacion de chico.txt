* `for`
el for es un bucle que se repite todas las veces que se quiera hasta que se diga que pare.

el for va primero el valor de la variable luego la cantidad de veces y luego el contador todo separado por coma ej:

for (i=0 ; i< 50 (hasta que i sea menor de 50) ; i++) {

lo que se quiera hacer////
}

* `&&`, `||`, `!`


el  && es para poner en un if o en donde sea, significa "y" o "y mientras"

el || es para decir  "o"

y el ! siginifica no si se pone con != no igual a un valor, (o distinto de "x")
